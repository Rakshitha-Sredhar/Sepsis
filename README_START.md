# How to Start the Sepsis Detection App

## Quick Start

### Option 1: Double-click START_APP.bat
Simply double-click the `START_APP.bat` file in the project folder. This will:
- Install dependencies if needed
- Start the development server
- Automatically open the app in your browser at http://localhost:3000

### Option 2: Use Command Line
Open PowerShell or Command Prompt in the project folder and run:
```bash
npm run dev
```
or
```bash
npm start
```

The app will automatically open in your default browser at **http://localhost:3000**

## Features

- **Authentication**: Login/Signup system
- **Dashboard**: View patient statistics and records
- **Sepsis Detection**: Enter patient vitals to detect sepsis risk
- **Results**: View detailed analysis with model performance comparison
- **AI Recommendations**: Get AI-generated clinical recommendations

## Troubleshooting

If the browser doesn't open automatically:
1. Wait a few seconds for the server to start
2. Manually navigate to: http://localhost:3000
3. Check the terminal for any error messages

## Stopping the Server

Press `Ctrl+C` in the terminal window where the server is running.
